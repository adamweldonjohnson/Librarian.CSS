Librarian.CSS is a no-nonsense css library for sites with large bodies of readable content.

Designed for the "F"

Blogs, "about us pages," news feeds and more benefit from enhanced readability provided by Librarian.css. While the color scheme is mainly monochromatic, you'll find pops of color in calls to action - links, forms and navigation - to enhance end user participation.

Nobody forgot about us devs, however. Librarian.CSS features useful layout tools and is designed for easy syntactical markup using nothing but easy-to-remember class names. No more class="v-2-5-seven-thirds" to get something that is "three-quarters" of a page!


Take a look at these user features:
  • Wide margins
  • Generous line heights
  • Simple, classic font selection
  • Colors only on links
  • Responsive, mobile-first design
And, for us devs:
  1. Responsive
  2. Easy class names
  3. Easy grid layout system
  4. Easy nav bars you can use anywhere
  5. Classy forms
  6. Clean Tables
  7. Includes buttons!

How-To (incl'd snippets!)

Articles and Sections

Librarian.CSS allows devs to give readable content pre-formatted "sections" in "articles." Give a master container div the class of "content". Place "article" divs inside your content div. Place "section" divs inside your articles.

Yes, it's that simple. Check out the code below!

Titles

Why fuss with margins? Use the class "title" and "subtitle" to give your titles some breathing room.

Buttons

Just need a button? Grab one below. Give it a class name of button and add your own modifiers. Profit.

Blockquotes

Blockquotes are easy! Just put in a blockquote element and make sure to start your blockquote with a " symbol. All's taken care of for you!
